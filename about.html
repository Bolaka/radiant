Radiant – Business analytics using R and Shiny

Radiant code

Version: 0.1.85, Date: 2015-3-26

Built using Shiny by Rstudio. Code available on GitHub

Package dependencies

Package dependencies: car, gridExtra, GPArotation, psych, wordcloud, AlgDesign, knitr, ggdendro, lubridate, ggplot2, pryr, shiny, magrittr, tidyr, dplyr, broom, htmlwidgets, shinyAce, rpivotTable, radiant
